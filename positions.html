ANRF PMECRG Project â€“ Causal AI for XR Protocols (Advt. Coming Soon)

This ANRF-funded project focuses on creating causal, explainable, and adaptive AI-based protocols for Extended Reality (XR) services in 6G/7G networks. It aims to overcome current limitations in AI-driven scheduling by embedding causal inference into reinforcement learning algorithms.

  • Highlights:
    • Develop a real-time XR test-bed using USRPs, Insta360 cameras, and Meta Quest headsets
    • Simulate traffic using NS-3 and OMNeT++
    • Design provable, performance-guaranteed causal RL protocols
  • Research Tools: Causal Inference, Network Calculus, Queuing Theory, CRL
  • Targeted Applications: Telemedicine, XR Education, AI-based Scheduling
